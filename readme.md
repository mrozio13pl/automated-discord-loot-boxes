## Setup

1. Download this git repo.
2. Install [NodeJS](https://nodejs.org/) on your computer.
3. Open CMD in the directory where you have the repository installed.
4. Execute these commands:
    ```sh
    npm install
    npm start
    ```
5. Paste your discord token.
